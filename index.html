HW05: Stuylin'

Feb 10, 2021

Due [S.O.P.]

Estimated time required: <=30min

…

Step 0: Summon duckie, KtS, writing utencils. Establish comms with your new team. Open HTML resources in browser. Read this entire assignment before beginning.

Phase 2:

  1. Save your latest index.html to your local machine as talk.html.
  2. Open a browser tab/window for viewing your locally-stored webpage.
  3. Augment talk.html so that
    • it applies styling defined in “external style sheet” main.css
    • your new teammates’ info replaces the old

Phase 3:

  1. Write CSS code in main.css:
    • Modify at least three tags with different rules.
    • Include at least one use of the font property. Note that you can separate many of the font propery values to constituent properties (i.e. font-style, font-size…)
    • Include at least two properties in addition to font.
  2. When you are satisfied with your edits (or the deadline arrives), upload your talk.html and main.css files to github. (Both should be in the root of your web repo.)

Guidelines + PROTIPs:

  • Include heading in each code file.
  • Simplicity is divine.
  • Use inline comments to describe/clarify/expound/remind/etc.
  • Alt-TAB / Cmd-TAB is your friend; use liberally to switch between code editor and browser.
  • Ctrl-R / Ctrl-Shift-R / F5 are your friends. Reload in browser after each little change you make in code. Grow incrementally…
  • Bring hard copy of code to class. (Except in the time of the Rona)



think: SIMPLE. think: SMART.
